**Cream & Co.** is a modern, high-conversion landing page designed for a premium ice cream brand. This project features a clean, mobile-first interface that showcases artisanal flavors, crowd favorites, and subscription services with a focus on elegant UI/UX and smooth interactivity.

---

## ‚ú® Key Features

* **Mobile-First Responsive Design:** A fully adaptive layout that transitions seamlessly from a hidden hamburger menu on mobile to a multi-column desktop experience.
* **Scroll Reveal Orchestration:** Utilizes the ScrollReveal library to trigger coordinated entrance animations for headers, images, and cards, creating a premium feel.
* **Interactive Navigation:** A functional mobile toggle menu that updates icons (menu to close) dynamically using JavaScript.
* **Product Showcases:** * **Popular Grid:** A curated section featuring top-rated flavors with hover-depth effects.
    * **Most Selling Categories:** Detailed product cards highlighting descriptions, original pricing, and discounted rates.
* **Visual Storytelling:** Large hero sections and parallax-style banner imagery to drive brand engagement.
* **Subscription System:** A stylized call-to-action form designed for discount enrollment and customer retention.

## üì∏ Technical Highlights

### **Modern Layout Engine**
The project leverages **CSS Grid** for complex sections like the "Popular" and "Discover" grids. By using `grid-template-columns` and `repeat()` functions within media queries, the interface automatically reorganizes itself based on the viewport width (540px, 768px, and 1024px).

### **Dynamic Animation Sequencing**
Instead of static elements, the site uses a JavaScript-based `scrollRevealOption` object to maintain a consistent animation duration and distance. Elements are revealed with specific delays (ranging from 500ms to 2000ms) to guide the user's eye through the brand story.
